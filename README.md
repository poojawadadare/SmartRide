Smart Ride is a web-based transportation management system 
that allows users to book rides, calculate fares, and track ride status efficiently.
The system provides a simple and user-friendly interface for managing ride details.

**Features**
User Registration & Login

Book a Ride

Fare Calculation

Ride Status Tracking

View Ride History

Admin Dashboard

Manage Users & Drivers

**Technologies Used**

Frontend: HTML, CSS, JavaScript, JSP

Backend: Java (Servlet & JSP)

Database: MySQL

IDE: Eclipse

Version Control: Git & GitHub

**Installation**

Clone the repository:

git clone https://github.com/poojawadadare/SmartRide.git

Navigate to the project folder:

cd SmartRide

Install dependencies:

npm install

Run the project:

npm start

**Objective**

The objective of Smart Ride is to provide a digital solution 
for ride booking and transportation management, improving convenience and reducing manual effort.
